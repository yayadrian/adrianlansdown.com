  • Eventcester

    Status: Beta

    A website to help users to discover events happening in their local area.

    Created as part of Launch48 Leicster this site originally created in 48 hours but has since had a number of improvements made.

    Although the beta site contains only a few very early features the main portion of work so far has gone into importing all of the event data. With sources coming from RSS, iCal and JSON feeds.

    View the Beta Site

    Tools used: Node.JS, ExpressJS, Isotope, Masnoary, HTML5, CSS3, Importing RSS, Importing iCal
